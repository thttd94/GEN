# proxy-manager-v1

Cài qua GitHub trên router.

## 1) Đổi mật khẩu cài đặt
Mở `install.sh` và sửa dòng:

```sh
INSTALL_PASSWORD="CHANGE_ME_INSTALL_PASS"
```

## 2) Trên router chạy

```sh
git clone <repo-url>
cd proxy-manager-v1
sh install.sh
```

Script sẽ hỏi:

```sh
Enter install password:
```

Nhập đúng mới cài tiếp.
